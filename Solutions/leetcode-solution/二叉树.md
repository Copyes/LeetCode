### 介绍
这里主要是二叉搜索树的解题方法和二叉树节点判断的解题方法

### 模版

#### 二叉搜索树
```js
function isInBST(root, target) {
  if(root === null) return false;
  if(root.val === target) return true;

  if(root.val < target) {
    return isInBST(root.right, target) 
  }
  if(root.val > target) {
    return isInBST(root.left, target);
  }
}
```
